# Gridiron — Football Intelligence Terminal

A prototype fantasy football intelligence platform, currently running on real 2024 NFL season data (via nflverse, free/public).

## Status
- Static prototype (index.html) — no backend yet
- Real player stats for QB/RB/WR/TE/K + all 32 team defenses (2024 season)
- Search, grades, boom/bust rates, game logs — all computed from real data

## Next steps
- Weekly automated data refresh (once 2026 season starts)
- Real weekly matchup-based projections
- AI-generated narrative reports
